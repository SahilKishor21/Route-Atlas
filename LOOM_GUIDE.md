# Loom Walkthrough Guide

## Suggested 5-10 Minute Flow

1. Assignment summary and topic choice
2. Quick architecture walkthrough
3. UI tour on desktop and mobile responsive layout
4. State management using Zustand
5. Theme toggle demonstration
6. Show empty state, loading state, and error retry flow
7. Build and deployment readiness

## Demo Prompts To Use

- Plan a one-day Kyoto itinerary for first-time visitors
- How to move around Kyoto with buses and trains
- What are key etiquette rules in temples and shrines?
- Best places for sunset views in Kyoto
